Waarom Python?
==============

.. Plusplunten:

* Elegante en compacte taal
* Uitgebreide (standaard)bibliotheken (*batteries included*)
* `Interactief`_ gebruik mogelijk (bv. `rekenmachine`_, `IPython`_)
* Uitstekende `documentatie`_
* Open source
* Schaalt goed: *start small, grow big*
* Meest populaire taal volgens laatste `IEEE`_ en `PYPL`_ rangschikking,
  en op 4e plaats in `TIOBE index`_
* Gebruikt in `vele verschillende toepassingsdomeinen`_: web,
  wetenschappelijke wereld, desktop, onderwijs, software ontwikkeling, ...
* Steeds vaker gebruikt als eerste taal voor beginners
* *Fun*

.. _interactief: https://docs.python.org/3.6/tutorial/interpreter.html#interactive-mode

.. _rekenmachine: https://docs.python.org/3.6/tutorial/introduction.html#using-python-as-a-calculator

.. _IPython: https://ipython.org/index.html

.. _documentatie: https://docs.python.org/3.6/

.. _IEEE: https://spectrum.ieee.org/at-work/innovation/the-2018-top-programming-languages

.. _PYPL: https://pypl.github.io/PYPL.html

.. _TIOBE index: https://www.tiobe.com/tiobe-index/

.. _vele verschillende toepassingsdomeinen: https://www.python.org/about/apps/
